# 🚀 AI Calorie Planner

A C# console app that calculates calorie needs and provides AI-based fitness advice.

---

## 🔥 Features

* BMR & TDEE calculation
* Calorie deficit detection
* AI recommendations
* Weekly fat loss prediction
* Excel export

---

## 🧠 Example

```
TDEE: 2545 kcal
Calories: 2000 kcal
Deficit: 545 kcal
Fat loss: 0.5 kg/week
```

---

## ▶️ Run

Set environment variables:

OPENAI_API_KEY=your_key
USDA_API_KEY=your_key

Then run:

dotnet run
